¿Qué es?

El software especializado que ayuda a restaurar archivos eliminados de un dispositivo de almacenamiento digital

En Windows

En mi caso yo voy a descargar EaseUS Data Recovery para Windows que es una herramienta para recuperar esos archivos que sin querer , lo has elimidado permanente.Esta tiene unos requisitos de 32MB de espacio en el disco duro y 1 GB de RAM para un correcto funcionamiento. Nos metemos en esta pagina para descargar EaseUS Data Recovery y le damos al boton de descargar gratis

Una vez descargado el archivo .exe lo ejecutamos y seguimos el proceso de instalacion.(Este proceso puede tardar un poco).

Cuando la instalacion haya terminado , ejecute el programa para poder recuperar los archivos eliminados deseados

Y ya estaria

En Ubuntu

En mi caso yo voy a descargar R Studio, un recuperador de archivos eliminados .Para un correcto funcionamiento de la aplicacion hace falta 1 GB de RAM y al menos 1 GB disponibles en el disco duro. Para instalarlo hay que descargar el archivo desde esta pagina y darle a "Ubuntu" y se empieza a descargar el archivo

Abriremos una terminal y pondremos lo siguiente "sudo apt-get install r-base" Este proceso puede ser que dure un tiempo , pero despues de esto ya tendremos instalado R Studio

Comparativa

Ambas aplicaciones son de recuperacion de datos pero su funcion no varia pero la dificultad de instalacion es diferente en Ubuntu
